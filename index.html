Functionality
What is in index?
  • We are using leafletjs (Open source and free) to create the map view.
  • The map in the coordenates (13.789555279605084, -89.35745218093535).
  • By clicking we can add a bookmark.
  • By hovering we can see the location of the marker
  • We can drag the markers to new positions.
  • We can search for locations, and when we click the search button, the map moves to the location and places a marker.
  • We cannot access the index without first logging in, even if we change the URL to index.html
  • We can log out the index
What is in login?
  • We can log in after register
  • Incorrect credentials are validate
  • Fields are required
What is in register?
  • We can register a user
  • You can not register a user with the same username
  • We are using localstorage to save the users
  • Fields are required
